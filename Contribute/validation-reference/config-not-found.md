# <a name="config-not-found"></a>config-not-found

## <a name="error"></a>エラー

`Can't find config file '{file name}' at {docset path}.`

## <a name="resolution"></a>解決方法

ご利用の構成ファイルをリポジトリ内の期待される場所に確実に配置します。
